# 💊 PharmaGrow CRM (Frontend)

A modern **Customer Relationship Management (CRM)** system built using **React + Vite** to manage pharma business operations efficiently.

---

## 🚀 Tech Stack

* ⚛️ React.js (Frontend Library)
* ⚡ Vite (Fast build tool)
* 🎨 CSS / Tailwind (Styling)
* 🔗 REST API (Connected with Laravel Backend)

---

## 📌 Features

* 🔐 User Authentication (Login / Logout)
* 📊 Dashboard with analytics
* 👨‍⚕️ Customer / Client management
* 📦 Product / Order management
* 🔄 API integration with backend
* ⚡ Fast performance with Vite

---

## 🛠️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Bharatkabir/pharmaGrow-crm-FE.git
cd pharmaGrow-crm-FE
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Run the project

```bash
npm run dev
```

👉 App will run on:

```
http://localhost:5173/
```
